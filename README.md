# Storylink-BOB-frontend

#### Project Overview:

This project is a frontend application for Storylink, a platform for sharing and reading stories. It is built using React.js and includes various features for users to read, write, and manage their stories.

#### Steps Done:

1. **Project Setup:**
   - Initialized a new React.js project using `create-react-app`.
   - Structured the project directories and files.
   - Installed necessary dependencies.

2. **Component Development:**
   - Created components for Home, Login, Register, Single Story View, Write Story, Navbar, Footer, and Dashboard.
   - Developed responsive UI components using Bootstrap 5 for better mobile compatibility.
   - Implemented routing using React Router for navigation between different pages.

3. **Styling:**
   - Styled components and layouts using SCSS for better design and visual appeal.
   - Utilized SCSS variables and mixins for consistent theming and styling across the application.
   - Implemented dark mode theme toggle functionality.

4. **Functionality:**
   - Implemented user authentication features including login and registration.
   - Developed functionality for users to read and view stories.
   - Implemented a dashboard for users to manage their stories.
   - Integrated forms for writing and submitting new stories.
   - Implemented dynamic routing for displaying single stories.

#### Installed Dependencies:

- **React.js**: Frontend library for building user interfaces.
- **React Router**: Used for declarative routing within the application.
- **Bootstrap 5**: CSS framework for building responsive and mobile-first websites.
- **node-sass**: Dependency for compiling SCSS files to CSS.
- **axios**: Library for making HTTP requests from the frontend to the server.
- **react-icons**: Icon library for React components.
- **react-bootstrap**: React components for Bootstrap 5.
- **react-markdown**: Component for rendering Markdown content as React components.

#### Project Overview:

- **Features**: User authentication, story reading, writing, and management.
- **Technologies Used**: React.js, SCSS, Bootstrap 5, React Router.
- **Folder Structure**: Organized components, pages, styles, and assets into separate directories for better maintainability.
- **Routing**: Utilized React Router for navigation between different pages of the application.
- **Styling**: Customized styles using SCSS and Bootstrap 5 for responsive design.
- **User Experience**: Designed with a focus on intuitive user interface and seamless navigation.
- **State Management**: Utilized React hooks for managing component-level state.
- **API Integration**: Communicates with backend APIs using Axios for fetching and posting data.
